# Neural_Network_Course

This repository hosts my homework assignments and solutions for the Neural Network course taught by [Professor Safabakhsh](https://scholar.google.com/citations?hl=en&user=zFsdqo8AAAAJ&view_op=list_works&sortby=pubdate) at AmirKabir University( Fall 2022). Additionally, the repository includes translated versions in English, primarily focusing on implementation-related questions. Please note that abstract questions and answers are available in Persian.
